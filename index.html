TauberJS

A powerful alternative to NPM Scripts that helps you create CLIs in one file.

Get started

Get started quickly

It takes exactly one minute to get started with Tauber. Add it as a dependency, add a script, and you're done.

Extremely powerful

Tauber allows you to run any script, JS code, JS file, and bash on any command, all in YAML.

Add to any project

Tauber can be added to ANY JavaScript project in just a couple lines.
